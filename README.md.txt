# Peace FM Tamil - PWA

Replace icon-192.png and icon-512.png with your station logo images.

**Important:** If your stream URL is `http://sapircast.caster.fm:16574/ZE9eu` and your site is served via HTTPS (GitHub Pages), the browser may block that HTTP stream as "mixed content". You must either:
- Use an HTTPS stream URL (recommended), OR
- Use an HTTPS proxy to relay the audio.

Deploy:
1. Put files in repo root.
2. Push to GitHub.
3. In repo -> Settings -> Pages -> Deploy from `main` branch `/ (root)` -> Save.
4. Open the GitHub Pages URL (https://username.github.io/repo). Test the Play button.
5. On mobile, "Add to Home screen" to install as PWA.
